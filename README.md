# Intro-to-GitHub-Assignment
Description 
GitHub collaborative workflows, including branches, issues, and pull requests, are demonstrated in this repository. Using pull requests and tickets to manage code changes is a useful exercise.

Project Details
Course: CS3321
Author: Deepan Pun
Contributor: Zerric Stewart

Shell Script Task
The shell script in this repository must be modified to perform a specific action.
File: repo_script-1.sh
Goal: Update the script to echo "Hello World!"
Expected Output:
Plaintext
Hello World!

How to Run
To verify the script output, use the following commands:
Make the script executable:
chmod +x repo_script-1.sh
Run the script:
./repo_script-1.sh

Collaboration Rules
In order to finish the assignment following workflow must be followed:
Branching: All work must be done on a branch named with feature/.
Commits: Multiple commits must be merged in a single commit for a linear history.
merging: Before merging in main pull request need to be made.
